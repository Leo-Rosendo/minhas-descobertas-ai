## 🚀 Como usar

1. **Navegue** pela seção que deseja
2. **Leia** as instruções no topo de cada arquivo
3. **Preencha** os campos marcados com `(( ))` (quando houver)
4. **Cole** o comando/template no seu agente de IA
5. **Pronto** — ele configura tudo automaticamente!

---

## 📖 Legenda

| Símbolo | Significado |
|---------|-------------|
| `[ ]` | O agente preenche automaticamente — não mexa! |
| `(( ))` | Você preenche com sua informação antes de colar |
