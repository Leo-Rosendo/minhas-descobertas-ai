## ✨ O que você encontra aqui

Templates prontos para usar com o agente Orion (AIOS/AIOX).
Cada template é um comando que você cola direto no agente — ele faz o resto sozinho.

---

## 📂 Templates disponíveis

### 🧠 [Memória + Saudação + Auto-Save](./memoria-persistente/memoria-saudacao-autosave.md)
Configure 4 recursos poderosos de uma vez só:
- ✅ Memória persistente do projeto
- ✅ Saudação personalizada ao iniciar o agente
- ✅ Auto-save automático a cada 29 minutos
- ✅ Auto-save por inatividade de 10 minutos

> 🆓 Gratuito | ⚡ 1 comando | 🤖 O agente faz tudo sozinho

---

## 🚀 Como usar

1. Acesse o template que deseja
2. Leia a legenda no topo do arquivo
3. Preencha os campos marcados com `(( ))`
4. Cole o comando no agente Orion
5. Pronto — ele configura tudo automaticamente!

---

## 📖 Legenda

| Símbolo | Significado |
|--------|-------------|
| `[ ]` | O **agente** preenche automaticamente — não mexa! |
| `(( ))` | **Você** preenche com sua informação antes de colar |
